# Sexy Launcher 4.0

A minimalist, e-ink friendly launcher designed specifically for the **Mudita Kompakt**.

## New in Version 4.0
- **Top 10 Apps View**: A dedicated section for your most frequently used applications, making navigation even faster.
- **Birday Integration**: Seamlessly import and display birthday reminders from Birday app backups (supports both SQLite and JSON).
- **Enhanced Notification Badges**: Redesigned for high visibility on E-ink screens with larger, bolder indicators and high-contrast borders.
- **Internal Launcher Reset**: For a cleaner experience, the launcher automatically returns to the main Home view when the screen is turned off (while navigating within the launcher).
- **Direct System Shortcuts**: Quick access to "App Notification Settings" and "DuraSpeed" directly from the Extra Settings menu.
- **Improved Call Tracking**: More robust contact name resolution and real-time status updates for active and missed calls.

## Key Features
- **Mudita Integration**: Full support for call and message notifications from Mudita Kompakt's system apps.
- **E-Ink Optimized UI**: High-contrast, text-first design with hollow-dot status icons and battery indicators tailored for electronic ink displays.
- **Smart Notifications**: A dedicated notification screen that filters out system noise and focuses on communication (calls, messages) and active media.
- **Interactive Header**: Fast access to 12h/24h clock, battery status (with charging indicator), and essential system info.
- **Focus-Oriented Navigation**: Minimalist app drawer with alphabetical scroller and customizable favorite apps.
- **Privacy First**: Local-only processing of notifications, contacts, and call logs.

## Designed for Focus
Sexy Launcher provides a clean, fast, and intentional user experience, replacing clutter with essential information. It transforms your Mudita Kompakt into an even more powerful tool for digital well-being.
