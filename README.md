# Sexy Launcher

A minimalist, e-ink friendly launcher designed specifically for the **Mudita Kompakt**.

## Designed for Focus
Sexy Launcher provides a clean, fast, and intentional user experience, replacing clutter with essential information.

- **Mudita Integration**: Full support for call notifications from Mudita Kompakt's default phone app.
- **Minimalist Header**: A fullscreen, interactive status area with 12h/24h clock and e-ink optimized indicators.
- **Smart Notifications**: A dedicated screen that filters out noise and keeps you focused.
- **E-Ink Optimized**: High-contrast UI with hollow-dot status icons and text-first navigation.
